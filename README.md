A small frontend project built with **HTML, CSS, and vanilla JavaScript**.

This project presents a list of travel destinations with a clean layout and simple user interactions.  
It was created as a **practice and portfolio sample**, focusing on frontend fundamentals rather than a full production app.

## Features
- Semantic HTML structure
- Responsive card-based layout
- Sticky navigation with smooth scrolling
- Dark mode toggle using CSS variables
- Form validation with user feedback (error / success states)
- Active navigation links based on scroll position
- Back-to-top button for improved user experience

## Technologies Used
- HTML5
- CSS3 (Flexbox, CSS Variables, Transitions)
- Vanilla JavaScript (DOM manipulation, events, scroll handling)

## Purpose
The purpose of this project is to demonstrate:
- Clean and structured HTML/CSS
- Basic UI/UX patterns
- Practical JavaScript interactions without frameworks

This project is **not intended to be a complex or production-ready application**, but a clear and well-structured example of frontend fundamentals.
